# react-ecosystems-skeleton
---
react+redux+redux-saga+react-router-redux+immutable

## Recipes
* react
* redux
* react-redux
* react-router
* react-router-redux
* redux-saga
* immutable
* reselect
* normalizr


## Get Started

```node
npm install
```

### development

gulp dev
http://localhost:8080

### production

gulp build
http://localhost:3030
